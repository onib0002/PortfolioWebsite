This navigation is possible by utilizing the concept of Hypertext. It is basically a text which refers to other web pages.

CSS

I use the CSS language for describing the presentation of Web pages, including colors, layout, and font and different stylings. It allows one to adapt the presentation to different types of devices, such as large screens, small screens, or printers(responsiveness). CSS is independent of HTML and can be used with any XML-based markup language.
